알고리즘
	1. 연산자 배열을 만듬. ( 3 2 1 4 -> 0 0 0 1 1 3 4 4 4 4 )
	2. 순열 나열 방식으로 연산자 순열을 하나씩 만들고, 계산하고 최대최소 확인
	3. 단, 계산횟수를 줄이기 위해 순열을 만들때 연산자가 같은 것끼리는 교환하지말자(같은 자리는 교환해야함.)

다른 알고리즘
	DFS
	1. 연산자가 4개이므로, +가 0 이상일때 -> + 하나 줄이고, +를 배열에 추가 or 첫번째 수 계산
				-가 0 이상일 때 -> - 하나줄이고, -를 배열에 추가 or 첫번째 수 계산
				...
		-> 끝까지 다왔을 때, 연산자 배열에 맞추어 계산 or 리턴
