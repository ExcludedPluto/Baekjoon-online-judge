알고리즘 (구현)
	1. Map 구현 - {점수, next, blueNext} 구조체 배열로 구현. 총 33칸 있으니, 33개 배열로 선언하고 출발-도착점 포함하여 각 칸의 번호를 적절히 매겨서 맵을 구현
			blue는 없으면 -1로 지정.
	2. deep을 주사위 인덱스로 한 재귀를 실행
		-말을 순서대로, 현재 주사위 눈에 따라 옮김.
			-> 옮기다가 도착점에 도착하면 중지 -> 옮긴 후의 말 배열과 점수는 증가 없이, deep 1추가 하여 재귀 실행.
			-> 다 옮기고 도착점에 다른 말이 있는지 확인 -> 없으면 (옮긴 후 말 배열, 점수 추가, deep +1)하여 다음 재귀 실행
		-deep == 10이면 넘어온 점수를 리턴.
		-모든 재귀문 중 가장 큰 점수를 리턴.
	

주의점
	1. 구조체 초기화 시, map = { {1,1,1}, {2,2,2} , ...} 로 해야함 / map = { (1,1,1), (2,2,2), ... } 은 인식 못함.
